# Questão para descrição 0

## Tarefa

Explique como você encheria uma garrafa vazia em casa. Considere que você está em um cômodo diferente de onde você costuma encher.

## Erros possíveis

Trate os seguintes erros, descrevendo o que fazer se algum acontecer:

- Se a garrafa transbordar?
- Se a garrafa quebrar?
- Se a garrafa estiver suja?
- Se a garrafa já estiver cheia?
