## Como funciona o nível 1?

A **chave** para conseguir programar é **descrever** em um passo a passo uma tarefa específica para o computador.

Por isso, nesses exercícios você terá que descrever **em passos** uma atividade para uma **criança completamente leiga** no assunto, explicando tudo o que for necessário.

Lembre-se que, assim como uma criança inocente, o **computador irá executar tudo o que você ordenar**, então explique bem para que tudo ocorra com sucesso.

_Ah, essa etapa da programação é muito importante, então não se limite apenas às questões, pense em como você descreveria coisas que você faz no seu cotidiano._
