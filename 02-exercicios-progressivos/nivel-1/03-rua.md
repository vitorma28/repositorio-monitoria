# Questão para descrição 3

## Tarefa

Explique como atravessar uma rua com segurança.

## Erros possíveis

Trate os seguintes erros, descrevendo o que fazer se algum acontecer:

- Se houver carros passando.
- Caso haja sinaleira/botão para pedestre.
- Se o tráfego nunca diminuir o suficiente para atravessar.
